## 📌 Overview

This project is an implementation of the classic **Connect 4** game with a strong focus on **algorithmic problem-solving techniques**.

The goal is not just to play the game, but to explore how different approaches like **Greedy, Divide & Conquer, Backtracking, and Dynamic Programming** can be applied to decision-making in games.

---

## 🧠 Algorithms Implemented

### 🔹 Greedy Approach
- Selects the best immediate move  
- Fast and efficient  
- Does not guarantee optimal results  

### 🔹 Divide & Conquer
- Breaks the board into smaller subproblems  
- Solves recursively and combines results  

### 🔹 Backtracking
- Explores all possible game states  
- Guarantees optimal solution  
- High time complexity  

### 🔹 Dynamic Programming
- Stores previously computed states  
- Avoids recomputation  
- Improves performance significantly  

---

## 🎯 Game Description

**Connect 4** is a two-player strategy game played on a **6×7 grid**, where players alternately drop discs into columns.

### 🏆 Objective
- Connect **4 discs in a row**:
  - Horizontally  
  - Vertically  
  - Diagonally  

This is a classic **combinatorial game** with a large number of possible states, making it ideal for algorithmic exploration.
